## Display size

Images are shown in a **fixed square box** (`object-fit: contain`) so different source sizes look uniform in the grid and confirm modal.

**Recommended source:** 512×512 px, transparent PNG or WebP.

Legacy gift id `pizza` maps to `donut` for older records.
