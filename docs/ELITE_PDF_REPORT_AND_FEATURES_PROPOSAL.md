# مقترح: تقرير PDF وتحليلات خطة الإليت

## 1. تقرير PDF (ميزة جديدة مقترحة)

### الفكرة
إتاحة طباعة/تحميل **تقرير PDF** من داخل لوحة Business Pro (لحساب الإليت فقط) يجمع:
- ملخص نشاط الحساب
- حركة الأعضاء والمتابعين
- الدعوات والنشاط المرتبط بها
- التقييمات والمراجعات
- فترة زمنية قابلة للاختيار (مثلاً آخر 7 أيام، 30 يوم، 3 أشهر، سنة)

### محتويات التقرير المقترحة

| القسم | المحتوى |
|-------|---------|
| **غلاف/ملخص** | اسم النشاط، الفترة، تاريخ إنشاء التقرير |
| **ملخص تنفيذي** | عدد الأعضاء، الدعوات النشطة، معدل التقييم، أهم المؤشرات |
| **نشاط الأعضاء** | عدد الأعضاء الجدد في الفترة، قائمة آخر الأعضاء (اختياري)، توزيع بسيط إن وُجد |
| **الدعوات** | عدد الدعوات في الفترة، النشطة vs المنتهية، ربما تفاصيل آخر 10–20 دعوة |
| **التقييمات والمراجعات** | متوسط التقييم، عدد المراجعات، ربما آخر مراجعات (مختصرة) |
| **نشاط الحساب** | عدد الزيارات للبروفايل (إن توجد هذه البيانات)، أو أي متري آخر متاح |

### التنفيذ التقني المقترح

- **مكتبة:** `jspdf` + `jspdf-autotable` لإنشاء PDF من الواجهة (لا حاجة لسيرفر إضافي للـ PDF).
- **مصدر البيانات:** نفس الاستعلامات المستخدمة في `ProOverview` وبيانات إضافية من Firestore:
  - `users` (أعضاء المجتمع عبر العلاقات المعرّفة في المشروع)
  - `invitations` (حسب `restaurantId` = الحساب)
  - `reviews` (حسب `partnerId`)
- **الواجهة:** زر في قسم Overview أو قسم جديد "Reports" في Business Pro:  
  **"تحميل تقرير PDF"** → اختيار الفترة (آخر 7 / 30 / 90 يوم أو سنة) → توليد وتحميل الملف.
- **الصلاحية:** عرض الزر وخيار التقرير فقط عندما `subscriptionTier === 'elite'`.

### خطوات تنفيذ مختصرة

1. إضافة الحزم: `jspdf`, `jspdf-autotable`.
2. إنشاء خدمة/دالة (مثلاً `eliteReportService.js`) تجمع البيانات للفترة المختارة.
3. إنشاء دالة توليد PDF (مثلاً `generateEliteReportPdf.js`) تبني التقرير من البيانات المجمعة.
4. إضافة واجهة في Pro Overview أو صفحة "Reports": اختيار الفترة + زر "تحميل تقرير PDF".
5. ربط الزر بشرط الإليت فقط.

## 2. ميزات إضافية مقترحة لخطة الإليت

(يمكن تنفيذ جزء منها الآن والجزء الآخر في مراحل لاحقة.)

| الميزة | الوصف | أولوية مقترحة |
|--------|--------|----------------|
| **تقرير PDF (أعلاه)** | تقرير تحليلي قابل للتحميل لفترة محددة | عالية |
| **تصدير بيانات (CSV)** | تصدير قائمة الأعضاء أو الدعوات أو المراجعات كـ CSV | عالية |
| **لوحة تحليلات أعمق** | رسوم بيانية بسيطة (أعضاء عبر الزمن، دعوات عبر الزمن، توزيع التقييمات) داخل Business Pro | متوسطة |
| **تقارير دورية (لاحقاً)** | إرسال تقرير PDF شهرياً بالبريد (يتطلب Cloud Function + بريد) | منخفضة |
| **شهادات/شارات الإليت** | شارة قابلة للتحميل أو مشاركتها (صورة/PDF) "Elite Partner" للمشاركة خارج التطبيق | منخفضة |
| **دعم مخصّص** | الرابط الموجود في الـ config (supportLevel: 'dedicated') — التأكد من ظهور وسيلة تواصل مخصصة للإليت في الإعدادات أو صفحة المساعدة | متوسطة |

---

## 3. ما هو موجود حالياً في الإليت (للمرجع)

- لوحة Business Pro (سطح المكتب) للإليت فقط.
- عروض Premium دائمة (بدون انتهاء).
- معرض محسّن: حتى 20 صورة (في مقابل 6 للـ Pro).
- قسم Featured Posts في البروفايل.
- في الـ config: `analytics: 'full'`, `supportLevel: 'dedicated'`, `apiAccess: true`, `canRunAds: true`.

---

## 4. الخلاصة

- **أول خطوة عملية:** تنفيذ **تقرير PDF** للإليت مع اختيار الفترة وربطه بـ Pro Overview أو قسم Reports، باستخدام البيانات الموجودة (أعضاء، دعوات، مراجعات).
- بعد ذلك يمكن إضافة **تصدير CSV** ثم **لوحة تحليلات بسيطة** داخل Business Pro لتعزيز قيمة خطة الإليت.

إذا رغبت، يمكن البدء بخطوة تنفيذ تقرير PDF (هيكل الملفات، دالة تجميع البيانات، ودالة إنشاء PDF) في الكود خطوة بخطوة.
